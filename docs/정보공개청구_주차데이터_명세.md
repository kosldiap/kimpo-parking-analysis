# 정보공개청구 — 김포공항 주차장 입·출차 전수 데이터 명세

> 목적: 2016~2018 분석 데이터를 2019~2025로 연장하여 코로나 이후 수요 패턴 재학습 및 장래수요 예측에 활용.
> 2016~2018 제공분과 **동일 형식**으로 요청 → 기존 분석 파이프라인에 그대로 결합 가능.

## 1. 청구 대상기관
- 한국공항공사 (서울지역본부 / 김포국제공항 주차 관리부서)

## 2. 청구 데이터
**김포국제공항 주차장 입·출차 전수(全數) 거래내역** (주차관제시스템 raw data)

## 3. 대상 주차장 (전체)
- 국내선 제1주차장
- 국내선 제2주차장
- 국제선주차장
- 화물청사 주차장
- 국제선버스
- 항공지원센터

## 4. 대상 기간
- **2019년 1월 ~ 2025년 12월** (제공 가능한 최신월까지)
- *범위 축소 시 최소: 2023년 1월 ~ 2025년 12월* (코로나 이후 재학습 최소요건)

## 6. 형식·단위
- **거래(차량) 단위 전수** (집계표 아님)
- **월별 · 주차장별 파일 분리** (2016~2018 제공 방식과 동일)
- 파일형식: Excel(.xls/.xlsx) 또는 CSV

## 7. 협조 요청(승인 원활화)
- **차량번호 등 개인정보는 일절 요청하지 않음** — 주차권번호(주차 1건당 고유 일련)만으로 분석 충분 → 개인정보 비공개 사유 해당 없음
- 2016~2018 제공 시와 동일한 담당부서·형식이면 처리 용이

### (참고) 2016~2018 제공 데이터 형식
- 2016: 월별 전체주차장 통합 .xls (시트 분할)
- 2017~2018: 주차장별·월별 .xls / 국제선버스·항공지원센터는 반기별
- 입·출차일시, 요금, 주차권번호 등 위 항목 동일 구성
