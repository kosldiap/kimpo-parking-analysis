# 김포공항 주차수요 분석 보고서 (v1.0)

분석기간 2016~2018 · 대상 6개 주차장 · 주차관제 입·출차 전수 데이터 기반.

---

## 1. 데이터 규모·신뢰성
- 정제 후 **5,587,968건** 거래 (완전중복 45,169·구간외 3,071 제거).
- 실제 주차(요금>0) **4,496,280건** / 비주차 통행(요금0) **약 109만 건(19.8%)**.
- **공식자료 검증**: KAC 공식 월별 대비 재구성 **2017·2018 = 98% 일치**(신뢰성 입증), **2016 = ~75%**(연중 광범위 DB 누락 확인).

## 2. 비(非)주차 통행 — 핵심 데이터 품질 이슈
- **요금 0원 = 유예시간(공식 10분) 내 빠른 재출차** = 드롭오프/U턴/만차회차. (유료주차는 11~12분부터 과금)
- 비주차 비중: **화물 49.6%, 직원 67.4%**, 국내·국제선 약 3%.
- 미제외 시 화물·직원 수요가 2배 이상 과대계상 → **직원 구조형 모형 R²가 음수(-0.29)→ +0.42로 전환**된 결정적 정제.

## 4. 수요 패턴
- **시간대**: 국내선 **새벽 5시 폭발 피크**(이른 항공편), 화물 **업무시간 쌍봉(10·15시)**, 국제선 6·14시. → 동인이 유형마다 상이.
- **주차→항공 출발 2시간 선행**(교차상관 국내 r=0.84/국제 r=0.67). 김포 국제선은 단거리라 인천과 달리 2h.
- **상관**: 국내1·국제선·항공지원·버스 양상관(0.4~0.5, 항공여객 공통동인) / **화물 독립**(물동량 별개).
- 그림: `03_hourly_profile.png`, `11_lead_xcorr.png`, `01_daily_timeseries.png`.

## 5. 점유율·초과수요
| 유형 | 평균 점유 | 최대 점유 | 정원 | 최대 점유율 |
|---|---|---|---|---|
| 국내선 | 3,100 | 4,931 | 4,052 | **122%** |
| 국제선 | 607 | 1,160 | 1,200 | 97% |
| 화물 | 53 | 352 | 737 | 48% |
| 직원 | 5 | 43 | 676 | 6% |

- **국제선 97%로 점유 복원 방법론 검증**(정원과 일치). 국내선 122%는 주차대행 overflow = 초과수요.
- **국내선 만성 초과수요 3종 근거**: 점유 122% · 만차일수 국내1P 연 365일(공식) · 입차 탐색시간 2014 2:40→2015 5:26.
- **balking 분석**: 시간대 통제 시 점유율↑→요금0 회차↑(국내선 +유의, 소폭). 단 심한 미충족수요는 입구 회차(우회)로 빠져 과소관측. 그림 `10_occupancy.png`, `14_balking_vs_occupancy.png`.

## 6. 모형 성능
- **운영형(lag)**: 화물 0.944·국내 0.925·국제 0.914·직원 0.540 (XGBoost, 2018 holdout).
- **구조형(no-lag, 시나리오용)**: 화물 0.889·국제 0.867·국내 0.835·직원 0.406.
- **5-fold 시계열 CV**로 안정성 확인(국내·국제·화물 표준편차 0.01~0.05, 직원만 큼).
- 우열: XGBoost ≳ RF > Linear ≫ Prophet (Prophet은 자기상관 미사용).
- 외부데이터(항공·기상) 한계효용 작음(lag가 수요 흡수). 시간대 항공이 **국제선 구조형 0.82→0.87** 개선.
- 그림 `12_model_compare.png`, `13_cv_stability.png`, `09_feature_importance_ml.png`.

## 7. 종합 결론
1. **국내선 만성 초과수요**가 정량 입증 → 신축 정당성.
2. **유형별 개별 모형**이 타당(동인 상이).
3. **"여객→주차" 98% 검증** → 장래수요 추정 신뢰 기반(#7).
4. **데이터 정제가 모형보다 중요**(요금0 비주차 제외).
