# 김포공항 주차요금 변경 이력 (#9 가격탄력성용)

## 확정 — 2016년 12월 1일 시행 (출처: YTN 2016-11-11, key=201611111046172761)
- 김포 장기주차: 주중 10,000 → **15,000 (+50%)**, 주말 15,000 → **23,000**
- 사유: "장기주차 차량이 워낙 많아 당일 이용객 불편" = 본 분석의 장기주차 점유 96% 발견과 일치
- 동시: 하이패스 도입(2017-03), 1,000대+ 주차공간 확보 계획

## 확정 — 2018년 12월 시행 (출처: KBS 2018-11-16, ncd=4075358)
한국공항공사 주차 혼잡완화 대책, 2018-12 적용:
| 주차장 | 주중 | 주말 |
|---|---|---|
| 국내선1·2 + 국제선 | 15,000 → **20,000** (+33%) | 23,000 → **30,000** |
| 화물터미널 | 15,000 → **12,000** (-20%, 인하) | 23,000 → **18,000** |

동시 발표: **국제선 주차장 부지 400면 주차빌딩 신축(2020 목표) + 김포공항 교통센터 건립 검토**
→ 본 프로젝트(교통센터/주차빌딩 신축사업)의 공식 배경. 초과수요(만차122%) 대책으로 요금인상+신축 병행.

## 데이터에서 탐지된 일상한(daily cap) 추이 (국내선, 약1일 체류 중앙요금)
2016Q1~Q3 ~11,500 → 2016Q4 15,000 → 2017 16,500(주중/주말 혼합) → 2018Q4 **20,000**
→ 개정 최소 2회: ~2016Q4(→15k/23k), 2018-12(→20k/30k). 단시간(3~4h) 요금은 ~7,000 안정.

## #9 가격탄력성 — 타이밍 한계
- 2018-12 개정: 잘 문서화됐으나 **우리 데이터가 2018-12-31 종료 → after 1개월뿐 → 현재 데이터로 추정 불가**.
- 2016Q4 개정: after(2017~18) 있으나 before(2016초)가 DB누락으로 불신 + 정확일자 불명.
- **결론**: 요금변동은 존재. 단 **2019+/2023~25 데이터 확보 시** 2018-12 사건(국내선·국제선 +33% 인상, 화물 -20% 인하)으로 단절적시계열 추정 가능. **화물 인하**는 점유 검열(만차) 없어 가장 깨끗한 탄력성.
- 국내선/국제선 인상은 만차 검열로 관측수요 반응 둔화 → 탄력성보다 "피크 완화 효과" 관점.
