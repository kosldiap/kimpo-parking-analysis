# 김포공항 주차수요 분석 — 작업 정리 (2026-06)

## 0. 프로젝트 개요
- **대상**: 김포국제공항 6개 주차장(국내선1·2, 국제선, 화물청사, 국제선버스, 항공지원센터) 2016~2018 주차관제 입·출차 **전수 데이터(정제 후 5,587,968건)**
- **목표**: 시간당 주차수요 예측모형 구축 → 교통센터(주차빌딩) 신축사업 장래수요·편익 분석
- **저장소**: github.com/kosldiap/kimpo-parking-analysis (비공개). 원본데이터는 .gitignore 제외, 코드·문서·대표그래프만.

## 2. 데이터 품질 — 핵심 발견·처리
1. **주차권번호 9999999 제외**(허수).
2. **요금 0원 = 비주차 통행**(유예 10분 내 드롭오프/U턴/만차회차) → 주차수요서 제외. 미제외 시 화물 ~50%·직원 ~67% 과대. **직원 모형 음수→정상의 결정적 정제.** (공식 무료시간 10분과 일치)
3. **2016년 연중 ~22~28% 과소집계**(DB누락) → 수준값 신뢰낮음, 2017~18 위주.
4. **화물 2017-09 = 10월 중복파일** 등 완전중복 45,169건 제거.
5. **공식 검증**: KAC 공식 월별 대비 재구성 **2017·18 = 98% 일치**(국제선 105%=국제선버스 포함차이).

## 3. 모델링 결과 (시간당 입차, 4유형 개별)
**2018 holdout R² (XGBoost)** / 5-fold CV
| 유형 | 운영형(lag) | 구조형(no-lag) | CV(운영) |
|---|---|---|---|
| 화물 | 0.944 | 0.889 | 0.936±0.011 |
| 국내선 | 0.925 | 0.835 | 0.908±0.023 |
| 국제선 | 0.914 | 0.867 | 0.874±0.047 |
| 직원 | 0.540 | 0.406 | 0.506±0.082 |

- **두 체제**: 운영형(lag, 단기예측 최강) / **구조형(lag 제외, 미래 시나리오용)**.
- 우열: **XGBoost ≳ RF > Linear ≫ Prophet**(Prophet은 자기상관 미사용으로 열위).
- 외부데이터(항공·기상) 한계효용 작음(lag가 수요 흡수). 시간대항공이 국제선 구조형만 개선(0.82→0.87).
- 직원은 자체 성장추세라 예측 한계.

## 4. 인사이트
**① 국내선 만성 초과수요** — 점유 최대 122%, 만차 연 1,018h(≈42일), 설계기준(연30위) 점유 115%, 만차일수(공식) 국내1P 연 365일, 입차 탐색시간 2014년 2:40→2015년 5:26. 타 유형은 만차 0h → 국내선만 구조적 부족. **신축 정량근거.**

**② 유형별 특성 상이** — 시간대(국내 새벽5시/화물 쌍봉), 회전율(화물 1.15 vs 국내 0.57 점유형), 명절배율(여객 0.85~1.0 무관/화물·직원 0.23~0.33 급감), 게이트 합피크 18시(≠입차 5시).

**③ 여객↔주차** — 공식 대비 98% 일치, 주차가 항공 출발 2시간 선행(교차상관, 김포 국제선도 단거리라 2h).

**④ 자율주행/카셰어링 취약성** — 공항 여객주차 점유의 **96%(국내)/94%(국제)가 다일 여행주차**=카셰어링 제거가능. 화물42%/직원28% 견고. 카셰어링 40% 전환 시 여객주차 ~40% 감소 시나리오. → 주차빌딩 좌초자산 위험.

**⑤ 매출·감면** — 국내선 감면율 15.1%(화물21%·직원5%), 건당수입 국내선 20,949원 vs 국제선 10,606원.

## 5. 외부 맥락 자료 (수집·검증)
- **항공통계**: 일별(항공정보포털)·시간대별(KAC) 운항/여객/화물. 항공사 LCC vs FSC: **2016년 이미 LCC 51% 성숙 → 2016~18 불변(null finding)**.
- **기상**: 기상청 ASOS 서울108 시간자료.
- **시설현황**: 실제 정원(국내선4052·국제선1200·화물737·직원676).
- **요금 변경 이력**: 2016-12-01 주중 10k→15k(+50%), 2018-12-01 15k→20k(+33%, 화물 -20%). 둘 다 "장기주차 과다→혼잡" 대책 + 400면 주차빌딩/교통센터 신축 발표(=본 프로젝트 대상). 우리 분석(장기점유96%·만차122%)이 정책근거와 정확히 부합.
- **2024 항공여객 이동특성 조사(공개보고서)**: 김포 접근 지하철34.5%·택시32.2%·자가용~20%대. ⚠️ 2017설문 자가용56%와 비교는 **교란 3종(철도 골드라인2019·서해선2023 개통 / OD·거리 구성 / 표본차)** 으로 AV 증거 아님. 철도전환(실증)+AV(시나리오) 이중구조로 정리.
